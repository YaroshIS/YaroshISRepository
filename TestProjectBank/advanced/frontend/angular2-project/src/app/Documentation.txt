Документация к проекту. Задача: вывод курсов валют в табличном и графическом виде.

Описание компонентов:
1. currency-counter - Основной управляющий компонент. Служит буфером для связи между остальными компонентами и сервисом currency-service. 
-Осылает запросы к сервису currency-service
-Приводит полученные данные к виду, в котором их можно будет передать сервису currency-service.
-Получает ответы от сервиса currency-service, приводит к нужному виду и передаёт остальным компонентам
-Хранит список всех валют

2. currency-form - дочерний компонент currency-counter, отвечающий за возможность выбора нужных валют.
-Принимает список валют от currency-counter.
-Выводит в список
-Позволяет выбирать нужные валюты
-Позволяет при нажатии на кнопку "Отправить" отсылает запрос к управляющему компоненту currency-counter.
-Позволяет очищать форму по нажатию кнопки

3. currency-graphics - дочерний компонент currency-counter, отвечающий за вывод курсов валют, отображаемых в таблице компоненте currency-list.
-Выводит график полученных валют
-Хранит переменную, отвечающую за максимальное количество дат на графике
-Хранит период, на который нужно выводить графики

4. currency-list - дочерний компонент currency-counter, отвечающий за вывод курсов валют в табличном виде
-Выводит таблицу.
-Хранит ID всех выбранных валют
-Хранит настройки вывода количества валют
-Имеет пагинацию

5. currency-service - сервис, служащий для связи между php-сервером и приложением.
-Принимает исходные данные, отправляет запрос к серверу, получает ответ и возвращает его управляющему компоненту.

Полное описание каждого коомпонента:
1. currency-counter:
-хранит переменные listComponent и graphicsComponent, через которые можно вызывать методы соответствующих компонентов.
-constructor(...) - конструктор, в котором получается экземпляр сервиса currency-serive для получения/отправки данных

-ngOnInit() - запускается, когда компонент загрузился и готов к работе. Отсылает запрос к сервису на получение списка доступных валют

-GetSelectedCurrencies(...) - Обрабатывается каждый раз при нажатии на кнопку "Отправить"
 Принимает массив с а) выбранными валютами, б) начальной датой, в) конечной датой.
 Передаёт даты в компонент currency-graphcis
 Передаёт ID выбранных валют и даты в компонент currency-list

-GetRates(...) - обрабытывается при переключении страницы в компоненте currency-list или при нажатии на ссылку "Show More Dates".
 Метод отправляет данные как в currency-list, так и в currency-graphics. Для того, чтобы график перерисовывался только при смене страницы, переданные ID сравниваются с теми, что уже есть на графике. Если ID совпадают - в компонент currency-graphics данные не передаются. В противном случае - данные передаются и в currency-graphics, после чего он перерисовывается.
 Т.к. результат запроса не содержит имени валюты (только ID), в каждый первый элемент массива, содержащего курсы валюты на запрошенный период, записывается имя валюты. 

-GetRatesToGraphics(...) - обрабатывается, когда график посылает запрос на получение курсов валют на период.

-CurIDtoName(...) - метод, преобразующий ID валюты в её имя.

***


2. currency-form: 
-хранит следующие переменные:
 currencyList - принимаемый от currency-counter список доступных валют.
 addList - переменная, с помощью которой можно вызвать метод currency-counter.
 selectedCurrencies - массив выделенных валют
 currencyFirstDate, currencySecondDate - переменные, хранящие даты в строковом формате(yyyy/mm/dd)
 showList - переменная, отвечающая за отображение списка валют
 error - объект, отвечающий за наличие и содержание ошибки. Имеет два поля: formValid, msg
 
В конструкторе инициализируем начальные значения

ToggleShowList() - переключает отображение списка валют
 
AddCurrency(currency) - добавляет/убирает выбранную валюту currency из массива выбранных валют

SendCurrencies() - проверяет валидность введенных данных и отправляет данные в currency-counter.

ClearSelectedCurrencies() - очищает форму
***


3. currency-list:
-хранит следующие переменные:
 getRates - переменная, которая отсылает нужные ID и даты управляющему компоненту currency-counter.
 currencies - двойной массив, хранящий курсы каждой валюты на выбранный период
 selectedID - массив, хранящий ID всех выбранных валют
 currentID - массив, хранящий ID отображаемых валют
 dateFrom - начальная дата
 dateTo - конечная дата
 dateToCurrent - дата, которая ограничивает количество выводимых дат
 COUNT_OF_COLUMNS - количество столбцов таблицы (кол-во выводимых валют)
 DATE_REPIOD - период, на который выводится курсы валют за один запрос

constructor() - инициализирует переменные и задает переменные  COUNT_OF_COLUMNS и DATE_PERIOD

GetSelectedCurrencies(selectedID, dateFrom, dateTo) - принимает ID выделенных валют и даты.
 Вычисляет количество страниц
 Вычисляет нужные ID
 Вычисляет начальный период времени
 Отправляет данные к управляющему компоненту
 
 
GetRatesOnPage(pageNumber) - обрабатывается при смене страницы в таблице. 
 Обнуляет массив currencies
 Вычисляет, на какой период нужно взять курсы
 Вычисляет, на какое количетсво валют отправлять запрос

GetRates(rates) - метод, принимающий данные от управляющего компонента currency-counter.
 Если в массиве currencies же находились какие-то значения - добавляет rates в конец.
 В противном случае - заполняет массив currencies значениями rates

addDaysToDate(data, days) - прибавляет к дате data days дней.
***


4. currency-graphics:
-хранит следующие переменные:
 chartEl - переменная, используемая для вывода графика
 getRates - переменная, отсылающая ID нужных валют и начальную и конечную даты.
 currencies - массив получаемых курсов валют
 cursID - массив ID валют, отображаемых на графиках
 dateFrom, dateTo - начальная и конечная даты
 opts - настройки для графика
 COUNT_OF_RATES - количество максимальных значений на графике
 _chart - переменная для работы с графиком.

ngAfterViewInit() - обрабатывается после инициализации шаблона компонента
 настраивает график

SendRequest() - отправляет запрос на получение курсов выбранных валют на период с начальной даты по конечную

GetDates(dateFrom, dateTo) - принимает период, на который нужно совершать запросы

GetSelectedID(cursID) - принимает ID валют, на которые нужно совершать запросы

GetRates(rates) - принимает значения курсов валют и отрисовывает их в график
